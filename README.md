# My Linux Dotfiles
These are most of my custom configs. I use hyprland with [HyDE](https://github.com/prasanthrangan/hyprdots) as my main system theming. 

Included are my custom waybar modules (really nothing fancy) and modification to the default `wbarconfgen.sh` script. My script allows for seperate bars on 2 monitors. As it is, the solution isn't very elegant, but it works. The name of your monitors would need to be replaced in the modules. A `&` seperates which modules appear on which screen. See my [config](/waybar/config.ctl) for an example.
